# Gestión del Proyecto con Trello

Para gestionar el desarrollo de **Luciano's Boutique**, empleamos **Trello** como nuestra principal herramienta de seguimiento Kanban.

## Columnas de nuestro Tablero
1. **To Do (Lista de tareas)**: Aquí reside nuestro Backlog. Tarjetas como "Crear la página 404" o "Configurar Express" esperan a ser priorizadas.
2. **In Progress (En proceso)**: Tareas que el equipo está desarrollando en este momento (ej. "Añadir persistencia al CartContext").
3. **Review (Revisión)**: El código está terminado, pero necesita una revisión de pares (Code Review) para asegurar que la interfaz cumple con los estándares minimalistas de la boutique.
4. **Done (Hecho)**: Funcionalidades fusionadas en la rama principal y listas para producción.

Este sistema visual nos permite identificar cuellos de botella y mantener un ritmo de desarrollo saludable.
